| #   | Task                                                          | File                   | Status  |

| 3   | Security docs in README                                       | `README.md`            | Pending |

**Notes:**

- **No data-dependent branches**: all `if`/`else` replaced with `@intFromBool` + bit masking
- Padding detection via `cN == 0xFE` mask without branching
- Invalid char detection via `@intFromBool(cN >= 0x40) & ~@intFromBool(cN == 0xFE)` — branch-free
- Output bytes at padding positions zeroed via `&=(pad -% 1)` trick
- Error accumulation via `err |= ...` — single check at end
- 8 new Zig tests, 6 new JS tests — all pass
- JS wrapper: `codec.base64.decodeConst(data, options?)`
